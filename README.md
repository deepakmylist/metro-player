**Metro Player** is a sleek, responsive, and feature-rich web media player. Built with modern web technologies and hosted reliably on Render, it provides a seamless playback experience with intuitive controls, playlist management, and a clean user interface.

## ✨ Key Functionalities

*   **Seamless Media Playback:** Smooth and responsive audio/video playback with minimal buffering.
*   **Intuitive Core Controls:** Standard media controls including Play, Pause, Next, Previous, Fast Forward, and Rewind.
*   **Volume & Seek Optimization:** Interactive progress bars for seeking track positions and adjusting volume dynamically.
*   **Playlist Management:** Easily queue, skip, and manage lists of tracks. 
*   **Responsive UI/UX:** A mobile-first design that looks great on desktops, tablets, and smartphones.
*   **Dynamic Metadata Display:** Displays current track information (title, artist, album art, duration) in real-time.
*   **Continuous Play:** Auto-plays the next track in the queue once the current media finishes.

## ⚙️ Getting Started

Follow these instructions to get a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

*   [Node.js](https://nodejs.org/) (v14 or higher recommended)
*   npm or yarn package manager
*   Git

### Installation

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/deepakmylist/metro-player.git](https://github.com/deepakmylist/metro-player.git)
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd metro-player
    ```
3.  **Install the dependencies:**
    ```bash
    npm install
    # or
    yarn install
    ```
4.  **Set up environment variables (if applicable):**
    Create a `.env` file in the root directory and add your specific configuration variables (e.g., PORT, API keys).
5.  **Start the development server:**
    ```bash
    npm start
    # or
    npm run dev
    ```
    *The application should now be running on `http://localhost:3000` (or your configured port).*
